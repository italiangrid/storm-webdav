# The StoRM WebDAV service

The StoRM webdav service provides http/webdav access to resources shared on a filesystem.

## Build dependencies

You will need:

- git
- java 17
- maven 3

## Build instructions

    mvn package

## Running the service

See [StoRM WebDAV installation & operation guide](doc/storm-webdav-guide.md)
